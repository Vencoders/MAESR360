# MAESR360: Masked autoencoder-based 360-degree video streaming via multi-scale feature fusion(2024 VCIP)



# Requirements
- python = 3.8
- pytroch >= 1.3.0
- tensorboardX
- cv2
- scipy
- einops
- timm = 0.4.12
- tqdm

# Usage

## Datasets

Dataset of Comparison with Baselines:

- Gaze360: Physically unconstrained gaze estimation in the wild. In Proceedings of the IEEE/CVF International Conference on Computer Vision 2019.

Dataset of Transfer Learning Capabilities:

- 360-degree video head movement dataset. In Proceedings of the ACM MMSys 2017.
- A Dataset for Exploring User Behaviors in VR Spherical Video Streaming. In Proceedings of ACM Multimedia Systems (MMSys) 2017, Taipei, Taiwan, June 20-23, 2017.

## Train

run `train.py`

## Test

run `test.py`
